### 2. 动态风险得分 (DRS) 计算

**计算公式：**
`DRS = W * ( Σ (P_i * S_i) / N ) * 100`

其中：
- `W` = 资产权重 (0.95)
- `P_i` = 第 i 个漏洞的利用成功率
- `S_i` = 第 i 个漏洞的严重性系数 (严重=1.0, 高=0.8, 中=0.5)
- `N` = 漏洞总数

**步骤1：计算加权平均成功率**

| 支付逻辑篡改（负值） | 90% | 1.0 (严重) | 0.90 |
| 支付逻辑篡改（精度溢出） | 80% | 0.8 (高) | 0.64 |
| SQL注入 | 70% | 1.0 (严重) | 0.70 |
| NoSQL注入 | 75% | 1.0 (严重) | 0.75 |
| 重放攻击 | 95% | 0.8 (高) | 0.76 |

**加权平均成功率 = (0.85 + 0.90 + 0.64 + 0.70 + 0.75 + 0.76) / 6 = 4.60 / 6 = 0.7667**
